# vapor-basics
https://vapor.codes


# Install Toolbox

```bash
brew install vapor  
vapor --help
```

# New Project 
```bash
vapor new projectName -n
```

# Build & Run
```bash
cd projectName 
open Package.swift  
```
and just run code 
- You should see the Console pop up at the bottom of the Xcode window.

```bash
[ INFO ] Server starting on http://127.0.0.1:8080
```
